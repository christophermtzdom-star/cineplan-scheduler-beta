# CinePlan Scheduler — Design System v1.0

## 1. Identidad visual

CinePlan Scheduler debe sentirse como un software profesional de preproducción cinematográfica: oscuro, limpio, modular y visualmente cercano a herramientas como Movie Magic, DaVinci Resolve y Linear.

## 2. Paleta base

- Background: `#0F172A`
- Surface: `#111827`
- Surface 2: `#1F2937`
- Primary / Success: `#22C55E`
- Secondary: `#3B82F6`
- Warning: `#F59E0B`
- Danger: `#EF4444`
- Text: `#E5E7EB`
- Text secondary: `#94A3B8`
- Border: `#263241`

## 3. Tipografía

- H1: 32px / Bold
- H2: 24px / Semibold
- H3: 18px / Semibold
- Body: 16px / Regular
- Caption: 12px / Regular

## 4. Componentes principales

### Sidebar
Debe mostrar:
- Logo / nombre de CinePlan
- Navegación principal
- Proyecto activo
- Progreso compacto
- Versión de app

### Dashboard
Debe mostrar:
- Header
- Progreso del proyecto
- Acciones rápidas
- Estado del proyecto
- Guía rápida

### Tarjetas
Todas las tarjetas usan:
- Fondo `Surface`
- Borde `Border`
- Radio 14px o 16px
- Padding 20px
- Sombra suave

### Botones
- Primario: verde
- Secundario: oscuro con borde
- Peligro: rojo
- Hover: borde verde o azul

### Tablas
- Fondo oscuro
- Encabezados destacados
- Bordes suaves
- Texto compacto

## 5. Iconografía

Usar Material Symbols Rounded siempre que sea posible.

Iconos principales:
- Inicio: `home`
- Guion: `description`
- Breakdown: `checklist`
- Stripboard: `view_agenda`
- Plan de Rodaje: `calendar_month`
- Hojas de llamado: `assignment`
- Reportes: `bar_chart`
- Configuración: `settings`
